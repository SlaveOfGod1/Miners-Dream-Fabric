# Miner's Dream

A Fabric mod for Minecraft that revives the classic **Miner's Dream** item from the popular **OreSpawn** mod.

## What it does

Use the Miner's Dream to instantly clear a **11×70×4 tunnel** in front of you, removing only natural cave blocks (stone, deepslate, dirt, gravel, etc.) while preserving **ores**, **structures**, and **everything else**.

* Ores are left untouched so you can see every diamond, gold, and redstone vein
* Structures like mineshafts, chests, spawners, and trial chambers are preserved
* Torches are placed along the center every 7 blocks for visibility
* Fluids (lava/water) are sealed at the tunnel boundaries
